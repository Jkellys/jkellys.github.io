Software I use, gadgets I love, and other things I recommend.

I get asked a lot about the things I use to build software, stay productive, or buy to fool myself into thinking I’m being productive when I’m really just procrastinating. Here’s a big list of all of my favorite stuff.

Development tools

  • Sublime Text 4

    I don’t care if it’s missing all of the fancy IDE features everyone else relies on, Sublime Text is still the best text editor ever made.

  • VS Code

    I use VS Code for all of my TypeScript and JavaScript work. It’s great for debugging and has a ton of great extensions.

  • iTerm2

    I’m honestly not even sure what features I get with this that aren’t just part of the macOS Terminal but it’s what I use.

  • TablePlus

    Great software for working with databases. Has saved me from building about a thousand admin interfaces for my various projects over the years.

  • Firebase

    I use Firebase for all of my personal projects. It’s great for hosting, databases, and authentication.

Design

  • Figma

    We started using Figma as just a design tool but now it’s become our virtual whiteboard for the entire company. Never would have expected the collaboration features to be the real hook.

Productivity

  • Click Up

    We use Click Up for all of our project management. It’s a little overwhelming at first but once you get the hang of it, it’s incredibly powerful.
